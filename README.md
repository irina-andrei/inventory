# inventory
A Python program that will read from the text file inventory.txt.
Allows user to:

    ♦ 1 - Load the inventory file 
    ♦ 2 - Enter new product to inventory 
    ♦ 3 - View all products  
    ♦ 4 - Re-stock   
    ♦ 5 - Search for a product   
    ♦ 6 - Values of each stock item   
    ♦ 7 - Find product for SALE
